## Фишки библиотеки

* Позволяет удобно создавать новые почты для yandex
  - Имеет поддержку решения капчи по ключу от сервиса RUCAPTCHA.
* Кроссплатформенность
  - Windows, macOS и Linux, везде, где только можете завести вашего питона.
